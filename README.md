# 📋 Licenciamento Posto

Sistema de gerenciamento de licenciamento ambiental para postos de combustível.

## 🚀 Tecnologias

- **Python 3.12**
- **FastAPI**
- **SQLAlchemy**
- **PostgreSQL**
- **Pytest**

## 🔧 Instalação

\\\ash
# Clonar repositório
git clone https://github.com/fintelman77718-ux/licenciamento-posto.git

# Entrar no diretório
cd licenciamento-posto

# Criar ambiente virtual
python -m venv venv

# Ativar ambiente virtual
# Windows
venv\Scripts\activate

# Instalar dependências
pip install -r requirements.txt
\\\

## ✅ Testes

\\\ash
# Executar testes
pytest tests/ -v

# Com cobertura
pytest tests/ -v --cov=src
\\\
